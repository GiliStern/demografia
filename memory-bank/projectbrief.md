# Memory Bank: Project Brief

## Project Overview
Demografia is a satirical Israeli-themed survival game built with React Three Fiber.

## Technology Stack
- **Framework**: React 18.2.0
- **3D Library**: React Three Fiber (@react-three/fiber)
- **Physics**: Rapier3D (@react-three/rapier)
- **State Management**: Zustand
- **Build Tool**: Vite
- **Language**: TypeScript
- **Testing**: Vitest
- **Documentation**: Storybook

## Project Structure
- React Three Fiber application
- TypeScript configuration
- Vite build system
- Storybook integration

## Key Dependencies
- @react-three/fiber: ^8.16.8
- @react-three/drei: ^9.114.0
- @react-three/rapier: ^1.4.0
- three: ^0.160.0
- zustand: ^5.0.9
